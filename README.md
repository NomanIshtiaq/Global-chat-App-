# Global Chat App

A real-time cross-platform chat application built with Flutter and Firebase.  
It allows users to register, login, and chat instantly using cloud-based messaging.

---

## Features

- Firebase Authentication (Login / Signup)
- Real-time messaging using Firestore
- User profile management (view & edit profile)
- Multi-screen navigation (Dashboard, Chat, Profile)
- Provider state management
- Cloud-based backend (Firebase)
- Clean and responsive UI

---

## Tech Stack

- Flutter (Dart)
- Firebase Authentication
- Cloud Firestore
- Provider (State Management)
